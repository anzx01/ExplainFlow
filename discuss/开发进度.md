## 迭代记录

### 2026-05-16 — MVP 基础版完成

- Graph → Storyboard → Render → Download 全链路 e2e 通过
- 渲染耗时约 96s（60s 视频，2 并发）
- 修复：ArrowConnect 缺少 from/to props 导致 `Cannot read properties of undefined (reading 'x')`

### 2026-05-16 — 手绘动画升级

**新增功能：**

- **真实字形描绘**：`server.mjs` 用 opentype.js 将文字转为 SVG path，Remotion 端用 `evolvePath()` 沿字形轮廓逐笔描绘（拉丁字符）
- **中文 fallback**：Caveat 字体不含中文，中文字符自动 fallback 到 scaleX 逐字划出动画
- **手的精确跟随**：`getPointAtLength()` 从实际 SVG path 取笔尖坐标，替代原来的估算位置
- **图形手绘抖动**：`jitterPt()` 用确定性 `Math.sin()` 对矩形/箭头路径点加轻微抖动，帧间一致不闪烁
- **标题手写动画**：场景标题也走 `SketchText` 手写动画（有 svgPath 时）
- **生成进度显示**：前端 RenderButton 显示阶段标签（合成语音 / 生成插图 / 渲染视频）+ 进度条

**关键技术：**

- 字体文件：`apps/render/public/fonts/Caveat.ttf`（1034 glyphs，含拉丁/西里尔）
- 新增依赖：`opentype.js ^2.0.0`、`@remotion/paths ^4.0.462`
- `.notdef` 字形检测：逐字符处理，跳过字体不含的字符，避免 NaN 污染 SVG path

**修复的 Bug：**

- `Malformed path data: M was expected to have numbers afterwards`：opentype.js 对缺失字符返回含 `LNaN` 的 `.notdef` 路径，改为逐字符处理并跳过 `.notdef` 字形

## 启动方式

```bat
scripts\start.bat   # Windows 一键启动
```

或分别启动：

```bash
bash scripts/dev-api.sh     # http://localhost:8000
bash scripts/dev-web.sh     # http://localhost:3000
bash scripts/dev-render.sh  # http://localhost:3001
```

日志输出到 `logs/`（api.log / web.log / render.log）。
