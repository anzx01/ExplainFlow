## 当前技术方法

### 1. 总体生成链路

```text
用户主题/参考图
  -> Storyboard
  -> TTS
  -> LLM/本地编译器生成自包含 Remotion TSX
  -> Remotion bundle
  -> renderMedia 输出 MP4
```

关键约束：

- Remotion 动画必须由 `useCurrentFrame()`、`interpolate()` 和 `Sequence` 驱动。
- 生成代码必须导出 `GeneratedVideo`。
- 只允许导入 `react` 和 `remotion`。
- 只允许受控 `staticFile()`：手部图片和 job-local 参考图资产。
- 禁止任意 `fs`、网络 fetch、模板库、组件库、CSS animation、SVG animate。

### 2. LLM 连通性与失败保护

- 渲染前通过 `/health/llm?force=true` 检查 DeepSeek/OpenAI-compatible 接口。
- 如果 LLM 不可用，立即停止，不继续 TTS、codegen、bundle 或 render。
- `OPENAI_BASE_URL`、`LLM_MODEL`、`CODER_MODEL`、`OPENAI_API_KEY` 可放在 `.env` 或进程环境变量中；日志不回显 key。

### 3. 原图逐笔揭示方案

核心思想：保留原图线稿像素质量，用路径遮罩逐步揭示原图，而不是把图重新矢量化后重画。

实现步骤：

1. `sharp` 读取本地参考图，使用宽容模式处理截断 JPEG。
2. 去除黄色/纸张背景，将深色线稿转成透明 PNG。
3. 从 mask 中提取深色区域，清理小噪点。
4. 使用骨架化算法得到笔画中心线。
5. 按端点、交叉点、闭环和空间顺序切分路径。
6. 控制路径数量约 120-150 条，避免太碎导致手乱跳。
7. 为每条路径估算 `revealWidth`，遮罩宽度覆盖原图笔画但尽量不误揭示相邻元素。
8. Remotion 中用 SVG `<mask>` 和 `strokeDashoffset` 逐条打开透明线稿图。
9. `HandPen` 使用同一批 `drawOps` 和 `pointOnPolyline()`，让笔尖沿当前笔画移动。
10. 末尾淡出 masked SVG 图层，并淡入 HTML `<Img>` 最终图层，避免透明 PNG 在 SVG 中出现黑底或双重加深。

效果收益：

- 中文不再是粗空心字。
- 圆圈、箭头、连接线保留原图锐利边缘。
- 首帧几乎空白，中间帧逐步出现，末帧接近原图。
- 画笔不是简单滑动，而是沿元素中心线移动。

### 4. 编码与画质

- Remotion 先输出 PNG 帧，避免中间帧压缩损失。
- MP4 使用 H.264，默认 CRF 8。
- 默认 `pixelFormat: "yuv444p"`，比 `yuv420p` 更适合高对比黑白线稿。
- 参考图资产默认生成到最终展示尺寸附近，减少浏览器二次缩放带来的边缘偏差。

### 5. 前端进度与预计剩余时间

- 任务阶段：`queued`、`tts`、`imagegen`、`codegen`、`bundling`、`rendering`、`done`。
- 前端根据阶段权重、当前进度、故事板总时长和已耗时估算剩余时间。
- `/storyboard` 渲染按钮和 `/jobs` 任务页都展示预计剩余时间。

## 迭代记录

### 2026-05-17 — 原图逐笔揭示白板视频完成

- 新增 raster reveal 预处理：本地参考图 -> 透明线稿 PNG -> mask -> 骨架中心线 -> reveal strokes。
- Remotion 端使用 SVG mask 逐笔揭示原图线稿，手部画笔跟随同一批中心线路径。
- 增加 `RasterFinalOverlay`，末帧使用 HTML `<Img>` 收尾，解决 SVG 透明图黑底和双重加深问题。
- 修复 composition 时长大于场景时长导致尾帧黑屏的问题。
- 默认使用 PNG 帧、H.264 CRF 8、`yuv444p`，提升密集线稿边缘稳定性。
- 对截断 JPEG 使用 `sharp(..., { failOn: "none" })` 宽容读取。
- 本地参考图预处理失败时优先走本地图 fallback trace，不再误触发 Seedream 重新生成图。
- `outputs/images` 全量 36 张图 e2e 通过。

### 2026-05-17 — 生成体验增强

- 前端增加预计剩余生成时间。

- 前端渲染进度阶段更新为：合成语音 / 生成动画代码 / 编译 Remotion / 渲染视频。
